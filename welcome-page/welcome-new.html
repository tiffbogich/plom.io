It's time to change the way we do model-based science.

With PLoM.io you can query, fork, or contribute data streams and models and much more. Your collections can be private or public, or shared with the people you work with.

  • PLoM.io provides

    A new ecosystem for model-based science

    With the click of a button, PLoM.io allows you to start at the cutting edge of scientific activity. By 'forking' an existing model in the library, and leveraging the fully tested statistical library of inference methods, PLoM minimises the need to write code and build models from scratch. PLoM.io leverages the web to boost productivity, including an automated peer review process and crowdsourced problem solving.



  • Great for

    Saving work resources

    Save useful resources like GitHub repositories, code libraries, fonts, tools and services. Make notes writing Markdown. Find the things you need with full-text search.

  • Great for

    Company library

    Keep everyone on the same page and learn with intra-company reading list. Track things like press hits, competition or helpful knowledge in one library. Read, comment and highlight the things your read.

Increase efficiency. Solve problems collaboratively. Get feedback. Publish.
